Steps to Reproduce the Issue

  • Delete the last record from the store.
  • Delete the new last record from the store (what had previously been the next to last record).
  • Call store.rejectChanges()

In this example...

  • Select Marge, click "Remove Selected Record."
  • Select Homer, click "Remove Selected Record."
  • Click "Undo Changes"

You'll see an alert() box if an error occurs.
